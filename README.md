<h2>🛠️ Installation Steps:</h2>

<p>1. Install Dependencies</p>

```
yarn
```

<p>2. Start In Development</p>

```
yarn dev
```

<p>3. Start Production </p>

```
yarn build

# Serve using Preferred method
npx serve dist
```
